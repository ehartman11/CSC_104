Python Packages Index (PyPI)

Python offers a variety of built-in packages that greatly improve developers and programmers capabilities. Many are the products of the hardworking and deeply committed open source community aligned to help make Python the most robust, accessible, and versatile programming language in tech. These packages range from gaming to analytics, from finance to database usage, from web design to APIs. Below are just a few examples of the packages available for PIP installation from the Python Packages Index repository:


  • Django and Flask
  • Numpy and Pandas
  • Pycharm - IDE
  • Pyglet and Pygame
  • Tkinter and Pyplot, or MatPlotLib

PEPs

PEPs, Python Enhancement Proposals, are design documentation providing information to the Python community, or describing new features for Python or its processes or environment. PEPs provide a (relatively) concise technical specification of the feature and a rationale for the feature. This is the primary mechanism for proposing major changes to Python, and it provides a framework for the community to discuss and evaluate proposals.
